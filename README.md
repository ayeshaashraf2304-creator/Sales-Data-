# Sales-Data
Analysics of data

**##Project Overview**
This project processes a spreadsheet database of 1000 orders to track store revenue, evaluate regional sales performances, and identify trending product categories.

**##InteractiveDashboard**
👉 https://public.tableau.com/views/SalesPerformanceDashboard_17816941090740/Analysis?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link

**##Key Questions Answered**
1. What is the total revenue and total units sold for the year 2025?
2. Which product categories generate the highest revenue?
3. Which month creates the most revenue?

**##Tools And Tecnical Skills Used**
* **Spreadsheets**: Raw data format verification.
* **Tableau**: Interactive dashboard design, trend line graphing, and geospatial tracking.
